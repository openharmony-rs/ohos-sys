# OpenHarmony AbilityAccessControl (AccessToken) bindings

Raw FFI bindings to the OpenHarmony AbilityAccessControl NDK (`libability_access_control.so`) —
the system API for querying whether an application has been granted a named permission.

Available since OpenHarmony API-level 12.

C API reference: https://gitcode.com/openharmony/docs/blob/master/en/application-dev/reference/apis-ability-kit/_ability_access_control.md
