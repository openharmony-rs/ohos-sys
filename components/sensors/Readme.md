# ohos-sensors-sys

Rust bindings for the `sensors` module of OpenHarmony.
This crate should only be used on OpenHarmony (`target_env = "ohos"`).
The official C-API documentation is available [on gitcode][sensors native guidelines].

[sensors native guidelines]: https://gitcode.com/openharmony/docs/blob/master/en/application-dev/device/sensor/sensor-guidelines-capi.md
