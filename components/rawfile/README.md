# OpenHarmony rawfile bindings

The OpenHarmony rawfile API allows the user to access so-called `rawfile`s that are bundled
into the `hap` bundle.

C API reference: https://gitcode.com/openharmony/docs/blob/master/en/application-dev/reference/apis-localization-kit/capi-rawfile.md
