# OpenHarmony AssetStoreKit (Asset NDK) bindings

Raw FFI bindings to the OpenHarmony Asset NDK (`libasset_ndk.z.so`) —
AssetStoreKit, the system service for storing short sensitive data such as
account credentials, tokens and bank-card numbers (≤ 1024 bytes).

Available since OpenHarmony API-level 11.

C API reference: https://gitcode.com/openharmony/docs/blob/master/en/application-dev/reference/apis-asset-store-kit/_asset_api.md
