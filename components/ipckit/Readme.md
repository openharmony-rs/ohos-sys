# ohos-ipckit-sys

Rust bindings for the IPCKit framework of OpenHarmony.
This crate should only be used on OpenHarmony (`target_env = "ohos"`).
The official C-API documentation is available [here][IPCKit API reference].

[IPCKit API reference]: https://docs.openharmony.cn/pages/v5.0/en/application-dev/reference/apis-ipc-kit/ipc__kit_8h.md
